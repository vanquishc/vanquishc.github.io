#CSS选择器

  1. 标签选择器
    作用：根据制定标签名称，在当前界面中找到所有该名称的标签，设置属性
    格式：
    ‘’标签名称 {
    ‘’ 属性： 值;
    ‘’}
    注意点：
  • 标签选择器选中当前界面所有的标签，而不能单独选中某一个标签
  • 标签选择器无论嵌套多深都能选中
  • 只要是HTML中的标签都可以作为标签选择器（h/p/img/ul/li/input…）
  1. id选择器
    作用：根据指定的id找到对应的标签，设置属性
    格式：
    ‘’#id名称 {
    ‘’ 属性： 值;
    ‘’}
    注意点：
  • 每个HTML标签都有id属性，即每个标签都可以设置id
  • 同一个界面中id的名称不可以重复
  • id命名规范，只能由字母、数字、下划线组成，且不能以数字开头
  • id属性一般留给js使用，除非有特殊情况，否则不要给id设置样式
  1. 类选择器
    作用：根据指定的类名称找到对应的标签，然后设置属性
    格式：
    ‘’.class名称 {
    ‘’ 属性： 值;
    ‘’}
    注意点：
  • 每个HTML标签都有class属性，即每个标签都可以设置类名
  • 同一个界面里，class的名称可以重复
  • class命名规范和id一致
  • 类名专门给CSS设置样式的
  • HTML中每个标签可以绑定多个类名
    一般将一些公共设置等代码抽取到一个类选择器中，让标签和这个类选择器绑定减少代码冗余
  1. 后代选择器
    作用：找到指定标签的所有特定后代标签，设置属性
    格式：
    ‘’标签名称1 标签名称2{
    ‘’ 属性: 值;
    ‘’}
    先找到所有名称叫做“标签名称1”的标签，然后在这个标签下面去查找所有名称叫做“标签名称2”的标签，然后设置属性
    注意点：
  • 后代选择器必须用空格隔开
  • 后代不仅指儿子，还有孙子…
  • 后代选择器不仅可以使用标签名，还可以使用别的选择器
  • 后代选择器可通过空格一直延续下去
  1. 子元素选择器
    作用：找到指定标签中的直接子元素，设置属性
    格式：
    ‘’标签名称1>标签名称2{
    ‘’ 属性: 值;
    ‘’}
    先找到所有名称叫做“标签名称1”的标签，然后在这个标签下面去查找“标签名称2”直接子元素设置属性
    注意点：
  • 子元素选择器只会查找儿子，不会查找其它被嵌套的标签
  • 子元素选择器之间需要用>符号链接，并且不能有空格
  • 子元素选择器不仅可以使用标签名称，可以使用其它选择器
  • 子元素选择器可以通过>符号一致延续下去
  1. 交集选择器
    作用：给所有选择器选中的标签相交的那部分标签设置属性
    格式：
    ‘’标签名称1标签名称2{
    ‘’ 属性: 值;
    ‘’}
    注意点：
  • 选择器和选择器之间没有任何链接符号
  • 选择器可以使用标签名称/id名称/class名称
  1. 并集选择器
    作用：给所有选择器选中的标签设置属性
    格式：
    ‘’标签名称1，标签名称2 {
    ‘’ 属性: 值;
    ‘’}
    注意点：
  • 并集选择器必须使用,来连接
  • 选择器可以使用标签名称/id名称/class名称
